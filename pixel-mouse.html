PIXEL MOUSE

Pixelmouse is a timed maze game created to test the quality of the game controllers. It is possible to play the game with just implementing the cardinal directions (WASD), but there are other buttons that you can implement to help you reach a a better time. The timer begins with the first instance of player movement, and teams will compete for the fastest average maze completion between 3 maze levels.

Maze Hotkeys

WASD : UP, LEFT, DOWN, RIGHT

Cardinal direction movement. Note that the arrow keys are not hotkeyed! Isn't it weird how hotkeyed is not spelled hotkeid like how paid is paid and not payed?

J : JUMP

Allows you to jump over barriers. The jump duration is 0.5 seconds.

R : RESET #1

Reset the timer on your current run and spawn back to the beginning. This button only works once! (By implementing both 'R' and 'T', you can reset up to 2 times per run.)

T : RESET #2

Reset the timer on your current run and spawn back to the beginning. This button only works once! (By implementing both 'R' and 'T', you can reset up to 2 times per run.)

Q : TOGGLE ROTATION

Toggle button that allows you to turn on and off a rotation setting. The rotation rate is 0.3 rev/sec. Useful to use with the 'F' key.

F : FORWARD KEY

Go forward based on your current direction. Useful to use with the 'Q' key.

Y : SET SPAWNPOINT #1

Set your spawnpoint to your current location. This button only works once! (By implementing both 'Y' and 'U', you can set a new spawnpoint up to 2 times per run.)

U : SET SPAWNPOINT #2

Set your spawnpoint to your current location. This button only works once! (By implementing both 'Y' and 'U', you can set a new spawnpoint up to 2 times per run.)
